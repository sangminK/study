```java
Iterable<Integer> iter2 = () -> 
	new Iterator<>() {
		int i = 0;
		final static int MAX = 10;
	
		@Override
		public boolean hasNext() {
			return i < MAX;
		}
	
		@Override
		public Integer next() {
			return ++i;
		}
	};
```
